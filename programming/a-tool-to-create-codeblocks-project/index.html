快速建立CodeBlocks工程的小工具

CodeBlocks是一个优秀的集成开发环境。不过我们初学者通常只用它写写简单的程序，这些程序往往只有一个源文件。直接打开Cpp源文件虽然可以编译，但无法加断点调试。如果要调试，则要建立一个工作空间和一个工程，步骤较为繁琐。今天写了个小程序，能快速为单一的源文件建立CodeBlocks工程，极大简化了小程序的调试工作。
